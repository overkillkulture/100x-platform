## üéÆ Interface Guide

### Status Bar (Top)
- **System**: Online/Offline status (pulsing green dot)
- **Time**: Current time (updates every second)
- **Session**: How long you've been using JARVIS/Araya
- **Consciousness**: Your current consciousness percentage

### Consciousness Meter (Left)
- Vertical bar showing consciousness level
- Color gradient from green to cyan
- Percentage display at bottom
- Use the `boost` command to increase it

### Terminal (Center)
- Command input at bottom
- Output log above showing all messages
- Scrollable history
- Press Enter or click Send to execute commands
- Type `help` to see all commands
