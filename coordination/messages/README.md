# Claude Inter-Instance Messages

This directory contains numbered message files for communication between different Claude instances.

## Message Numbering

Messages are numbered sequentially (001.md, 002.md, etc.)

## Message Format

```markdown
# Message [NUMBER]

**From**: [Claude Session ID]
**To**: [Target Session ID or "ALL"]
**Date**: [ISO timestamp]
**Priority**: URGENT / NORMAL / LOW

---

## Subject

[Brief subject line]

## Message

[Detailed message content]

## Action Required

- [ ] Task 1
- [ ] Task 2

## Context

[Any additional context or file references]
```
