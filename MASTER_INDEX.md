## ‚ö° TROUBLESHOOTING GUIDE

### **API Won't Start:**
‚Üí Read: `Desktop\API_PROBLEM_SOLVED.txt`
‚Üí Solution: Run `START_API_SERVER.bat` (kills zombie processes)

### **Platform Authentication Failed:**
‚Üí YouTube: Regenerate OAuth credentials
‚Üí Twitter: Delete `twitter_session` folder, re-login
‚Üí Instagram: Delete `instagram_session` folder, re-login
‚Üí Late API: Check API key in dashboard

### **Video Upload Failed:**
‚Üí Check file size limits (per platform in guides)
‚Üí Verify video format (MP4 or MOV)
‚Üí Compress with MoviePy if needed

### **Route 404 Error:**
‚Üí Multiple Flask servers running
‚Üí Solution: Use `START_API_SERVER.bat` (clean restart)
