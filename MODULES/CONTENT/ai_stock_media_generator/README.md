## üõ†Ô∏è TECHNICAL INTEGRATION

### Generate Image:
```python
from ai_stock_media import MediaGenerator

generator = MediaGenerator(api_key="your_api_key")

# Generate image
image = generator.generate_image(
    prompt="Professional business meeting in modern office",
    style="photorealistic",
    resolution="4k",
    aspect_ratio="16:9"
)

print(f"Image URL: {image.url}")
print(f"Generation time: {image.generation_time}s")

# Download
image.save("business_meeting.jpg")
```

### Generate Video:
```python
# Generate video clip
video = generator.generate_video(
    prompt="Ocean waves crashing on beach at sunset",
    duration=10,  # seconds
    resolution="1080p",
    style="cinematic"
)

print(f"Video URL: {video.url}")
video.save("ocean_sunset.mp4")
```

### Batch Generation:
```python
# Generate 50 variations
prompts = [
    f"Product photo of smartphone, angle {i}, white background"
    for i in range(50)
]

batch = generator.generate_batch(prompts, style="commercial")

print(f"Batch complete: {len(batch.images)} images")

for i, image in enumerate(batch.images):
    image.save(f"product_{i:03d}.jpg")
```
