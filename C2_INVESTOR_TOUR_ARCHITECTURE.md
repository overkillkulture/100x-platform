## üí° KEY ARCHITECTURAL DECISIONS

### 1. **Self-Contained Modules = Resilience**
Each showcase page can function independently. One breaks, others work.

### 2. **Progressive Disclosure = No Overwhelm**
GTA pattern: Show 6-8 choices max per level. Prevents paradox of choice.

### 3. **Existing Pages Stay Intact**
We organize them, don't rebuild them. Wrap in navigation, add context.

### 4. **Mobile-First Responsive**
All pages already have viewport meta tags. District grids use CSS Grid auto-fit.

### 5. **Metrics Everywhere**
Every district shows quantitative value: "186 modules", "92.2% accuracy", "C1√óC2√óC3=‚àû"

### 6. **Guided + Exploratory Paths**
